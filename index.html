BUUSecretFile

Posted on 2021-03-25

打开后出现一个很有意思的界面：

你想知道蒋璐源的秘密么？

当场疑惑，蒋璐源是谁？有啥子秘密？

二话不说当场F12，审查源码，发现 Archive_room.php:

访问后，中间有个SECRET超链接，点击进去后直接跳转到查阅结束。

肯定是有些东西藏在secret.php和end.php的跳转过程中

直接上burpsuite抓包，得到secr3t.php

访问后得到一段php源码：

secret

代码中的两个PHP函数：
strstr() ，查找一个子串是否在另一个字符串中存在，存在则返回子串以及后面剩余部分，区分大小写；

stristr() ,与 strstr() 函数作用相同，不区分大小写；

源码中提示 flag 在flag.php中，而要使得 include()将flag.php 包含进来，则要使得 前面的 if 不成立，而 if 不成立，需要满足4个条件：

file 变量不能含有 …/ 这样目录穿越的字符；

file 变量不能含有 tp，input，data 这几个字符；

既然 input，tq，data …/， 被禁用，那我们就无法进行目录穿越，或者shell写入连接蚁剑，所以想到使用 php://filter 这个php伪协议；

接下来构造payload：
secr3t.php?file=php://filter/read=convert.base64-encode/resource=flag.php

payload 解析：

php://fliter： 用于读取文件源码；

read： php://filter的参数，用于读取数据；

convert.base64-encode 将读取的PHP代码（数据流）进行base64 编码，否则php代码会被执行而不再页面显示；

resource：指向要读取的文件，（要筛选的数据流）

得到一段base64编码，解码后得到flag

BUUinclude1

Posted on 2021-03-25

拿到题目，发现有个tips

点击tips，出来一句话：Can you find out the flag?

根据本体的名字include以及url中的/?file=flag.php就能够知道这是一题文件包含题了。

用php伪协议读取flag.php的内容

php://filter 伪协议
该伪协议读取源代码并进行base64编码输出，不然会直接当做php代码执行就看不到源代码内容了。

payload为：file=php://filter/read=convert.base64-encode/resource=flag.php

出来一段base64编码：

PD9waHAKZWNobyAiQ2FuIHlvdSBmaW5kIG91dCB0aGUgZmxhZz8iOwovL2ZsYWd7YjQ4NzQzYmQtMDFhZi00ZTVkLWFiYTgtN2E0Yjc2NGEwMWE3fQo=

解码之后得到：
<?php
echo “Can you find out the flag?”;
//flag{b48743bd-01af-4e5d-aba8-7a4b764a01a7}

成功得到flag

最后补充一点php://filter伪协议的用法：

1.php://filter/read=convert.base64-encode/resource=[文件名]读取文件源码（针对php文件需要base64编码）

2.php://input + [POST DATA]执行php代码

发现有个hint.php，进去看一看，发现一句话：
flag not here, and flag in ffffllllaaaagggg
猜测flag应该在ffffllllaaaagggg里面
通过百度看别人的wp知道，这题是白名单验证
这道题定义了一个checkfile()函数

大致的功能如下:
1.定义了一个白名单：source.php和hint.php，接着判断$page是否为空，是否为字符串。
2.判断$page是否在白名单里，若在，则返回true
3.$_page是取出$page里面？前面的东西，然后再判断$_page是否在白名单里，若在，则返回true
4.如果上一步判断失败，则又考虑了url编码的问题，因为url在传入以后服务器会自动进行一次解码。因此传入二次编码后的内容，就可以使checkfile返回true。

这里涉及到了两个php函数：substr()和strpos()

1.substr()函数返回字符串的一部分。
使用规则如下：
substr(string,start,length)
string：必需。规定要返回其中一部分的字符串。
start：必需。规定在字符串的何处开始。
正数 - 在字符串的指定位置开始
负数 - 在从字符串结尾的指定位置开始
0 - 在字符串中的第一个字符处开始
length：可选。规定要返回的字符串长度。默认是直到字符串的结尾。
正数 - 从 start 参数所在的位置返回
负数 - 从字符串末端返回

2.strpos()函数查找字符串在另一字符串中第一次出现的位置（区分大小写）
使用规则如下：
strpos(string,find,start)
string：必需。规定被搜索的字符串。
find:必需。规定要查找的字符。
start:可选。规定开始搜索的位置。
